## C++ general notes
To get C++ compiling successfully on Mac, based on [stack overflow post on clang](https://stackoverflow.com/questions/79160879/iostream-not-found-with-g-and-clang-on-macos) had to run

```
sudo mv /Library/Developer/CommandLineTools/usr/include/c++{,-backup}
```
